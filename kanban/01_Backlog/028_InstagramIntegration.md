# Instagram Integration

## Descrição
Integração limitada com Instagram via browser automation. Leitura de notificações e mensagens (DM). Postagem programada de fotos e legendas. NOTA: API oficial da Meta requer app aprovado. Scraping pode levar a ban. Uso apenas para uso pessoal, baixo volume.

## Critérios de Aceitação
- [ ] Login via browser automation (instagrapi ou similar)
- [ ] Tool `read_instagram_notifications`: lê notificações recentes
- [ ] Tool `post_instagram`: foto + legenda → post (DM)
- [ ] Gerenciamento de sessão (2FA, cookies)
- [ ] Aviso de risco na UI sobre limitações e bans
