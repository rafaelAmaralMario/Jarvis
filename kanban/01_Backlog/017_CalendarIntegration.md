# Integração de Calendário (CalDAV)

## Descrição
Integrar CalDAV para ler e agendar eventos. Tools: `list_events` (eventos do dia/semana), `create_event` (adicionar evento), `update_event`. Suporte a iCloud, Google Calendar (via app password), Nextcloud.

## Critérios de Aceitação
- [ ] Instalar caldav lib
- [ ] Tool `list_events`: lista eventos em período
- [ ] Tool `create_event`: cria novo evento com título, data, duração
- [ ] Tool `update_event`: modifica evento existente
- [ ] Múltiplos calendários configuráveis
