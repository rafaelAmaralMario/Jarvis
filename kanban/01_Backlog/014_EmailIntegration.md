# Integração de Email (IMAP/SMTP)

## Descrição
Integrar email via IMAP (leitura) e SMTP (envio). Tools: `read_emails` (listar emails não lidos, buscar por remetente/assunto), `send_email` (enviar com anexos), `reply_email` (responder thread). Suporte a múltiplas contas.

## Critérios de Aceitação
- [ ] Tool `read_emails`: lista emails (INBOX, não lidos, busca)
- [ ] Tool `send_email`: envia email com texto formatado (HTML)
- [ ] Tool `reply_email`: responde a um email específico
- [ ] Suporte a anexos
- [ ] Múltiplas contas configuráveis
- [ ] Autenticação via app password (Gmail, Outlook, etc.)
