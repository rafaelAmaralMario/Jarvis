# Self-Improvement Module

## Descrição
Módulo que analisa o projeto (Project.md, git log, file tree), gera proposta estruturada com LLM (summary, priority, steps, risks, verification), executa passos com permissão por passo (ou unattended), streaming de progresso via bridge `_sistreams`.
