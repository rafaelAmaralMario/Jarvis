# Integração de Calendário (CalDAV)

## Descrição
Integrar CalDAV para ler e agendar eventos. Tools: `list_events` (eventos do dia/semana), `create_event` (adicionar evento), `update_event`. Suporte a iCloud, Google Calendar (via app password), Nextcloud.

## Critérios de Aceitação
- [x] Instalar caldav lib
- [x] Tool `list_events`: lista eventos em período
- [x] Tool `create_event`: cria novo evento com título, data, duração
- [x] Tool `update_event`: modifica evento existente
- [x] Múltiplos calendários configuráveis
