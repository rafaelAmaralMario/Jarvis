# T4: Workspace + Editor Unification

## Descrição
Unificar FileTree com ícones por tipo de arquivo, drag & drop entre pastas, busca unificada (arquivos + conteúdo + comandos), abas de editor arrastáveis, preview de imagem/PDF no editor. Melhorar a experiência de navegação no workspace.

## Critérios de Aceitação
- [ ] Ícones personalizados por tipo de arquivo no FileTree
- [ ] Drag & drop para mover arquivos entre pastas
- [ ] Busca unificada (Ctrl+P): arquivos, conteúdo, comandos
- [ ] Abas do editor arrastáveis para reordenar
- [ ] Preview de imagem e PDF no editor
- [ ] Indicador de arquivos modificados (dot) nas abas
