# 025: Auto-Update Runtime + App Icon + PyInstaller Build

**Date:** 2026-06-08

## Summary
Implemented the full auto-update runtime flow: download release asset → create updater batch script → replace executable/run installer → restart app. Created app icon. Fixed PyInstaller spec for CI build.

## Changes

### Auto-Update Runtime (`backend/jarvis/version.py`)
- `download_and_install()` now returns `{"success": bool, "path": str, "restart": bool, "message": str}` instead of just a path
- When running as frozen PyInstaller exe: creates `%TEMP%\jarvis_update.bat` that waits for PID to exit, then replaces exe or runs installer silently, then restarts
- Refactored: extracted `_fetch_releases()`, `_get_asset_for_platform()`, `_download_file()`, `_create_updater_bat()`, `_is_installer()` helper functions

### Bridge (`backend/jarvis/bridge.py`)
- `downloadAndInstall()` now returns dict with `restart` field
- Added `quitApp()` method — calls `webview.windows[0].destroy()` or `os._exit(0)` as fallback

### Frontend (`ui/src/components/Settings/UpdatesPanel.tsx`)
- Added restart flow: when `result.restart` is true, shows "🔄 Reiniciar JARVIS Agora" button
- Added `info` message type (blue styling)
- Uses `bridge.quitApp()` for restart

### Types (`ui/src/types/index.ts`)
- `downloadAndInstall` return type: added `restart?: boolean`, `message?: string`
- `JarvisBridge`: added `quitApp()` method

### App Icon (`scripts/jarvis.ico`)
- Generated by `scripts/generate_icon.py` using only stdlib (struct + zlib for PNG)
- 64x64 RGBA PNG in ICO container, glowing blue "J" on dark background
- Embedded in PyInstaller EXE (`Copying icon to EXE` confirmed)

### PyInstaller Build (`scripts/jarvis.spec`)
- Fixed `NameError: name '__file__' is not defined` — fallback to `sys.argv[0]`
- Build succeeds: `dist/JARVIS.exe` (~35MB)

### Other
- `scripts/build_exe.py`: use `sys.executable -m PyInstaller` for cross-platform
- `.gitignore`: added `dist/` to prevent committing build artifacts
- `ROADMAP.md`: updated with Fase 9 (auto-update + release)
