# Principios de UX

1. **Conhecimento em primeiro lugar** — o cerebro e o centro do sistema
2. **Atalhos de teclado** — power users navegam sem mouse
3. **Escuro como padrao** — tema dark first, claro como opcao
4. **Minimo, mas poderoso** — interface limpa, funcionalidades profundas
5. **Feedback constante** — toda acao tem reacao visual
