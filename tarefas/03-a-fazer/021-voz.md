## Descrição
Interface de voz com o assistente: Speech-to-Text (STT) via whisper.cpp e Text-to-Speech (TTS).

## Especificação Técnica

### C++ — VoiceManager
- STT: Integração com whisper.cpp (modelo pequeno .bin)
  - `voiceStartRecording()` → começa a gravar do microfone
  - `voiceStopRecording()` → para gravação, retorna texto transcrito
- TTS: Usar biblioteca C++ (espeak-ng, piper-tts) ou API
  - `voiceSpeak(text)` → reproduz áudio
- Eventos: `voice-recording`, `voice-transcribed(text)`, `voice-error`

### Bridge handlers
- `voiceStartRecording()` / `voiceStopRecording()` → grava e transcreve
- `voiceSpeak(text)` → fala texto
- `voiceIsAvailable()` → verifica se microfone/config OK

### React — Componentes
- Botão de microfone na StatusBar ou no AiPanel
- Indicador visual de gravação (pulsando vermelho)
- Indicador visual de fala (onda sonora)

### Atalhos
- Ctrl+Shift+M: toggle microfone
- Configurações: microfone padrão, velocidade TTS, idioma
